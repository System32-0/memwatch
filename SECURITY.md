# Security Policy
 
## Reporting a Vulnerability
 
If you find a security issue, feel free to open a public issue or reach out directly at **memwatch@code.cdespalo.com**. You can also use GitHub's **private vulnerability reporting** under the Security tab if you'd prefer.
 
Try to include what the issue is, where it shows up, and what you think the impact could be. I'll get back to you as soon as I can.
 
---
 
Only the latest version is maintained.
